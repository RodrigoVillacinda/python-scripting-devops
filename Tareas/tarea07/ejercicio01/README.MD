## Ejercicio 01 - Tarea01

Este programa en **Python** solicita al usuario ingresar calificaciones (**una por línea**).  
El ingreso de notas finaliza cuando se introduce el valor `-1`.  

Al finalizar, el programa muestra:

1. El **promedio de las notas** (redondeado a número entero).
2. Todas las notas en el **mismo orden en que se ingresaron**.

---

## 📝 Ejemplo de ejecución

### Entrada:
```

80
75
70
-1

```

### Salida:
```

75
80
75
70

````

> Explicación: el promedio de (80 + 75 + 70) / 3 = 75.  
> Se imprime primero el promedio y luego las notas en orden.

---

## ▶️ Ejecución

Clonar el repositorio y ejecutar el archivo `main.py`:

```bash
python main.py
````
